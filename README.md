These are various Tools I've made, primarily useful for modding Heroes of Might and Magic 3. 

CSV to JSON - this one changes a properly set up CSV file (Such as the example heroes file in same folder) into json files to be used for your mod. 

simple text finder - finds a given phrase in a folder and all subfolders. UTF-8 encoding by default. 

simple text swapper - replaces text in a folder (and, optionally, all subfolders) for another text. ASCII encoding by default. 

Object Analyzer - runs through EdObjects and info from Hota.dat to produce a detailed description of each object type in the game.
